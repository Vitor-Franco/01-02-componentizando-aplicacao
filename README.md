## 🚀 Getting started

Clone o projeto e acesse a pasta

```bash
$ git clone https://github.com/Vitor-Franco/01-02-componentizando-aplicacao.git && cd 01-conceitos-reactjs
```

Siga o passo-passo abaixo.

```bash
# Instale as dependências
$ yarn

# Rode a fake API
$ yarn server

# Start o projeto
$ yarn dev
```
